## 1. 概要

**Thorlabs DCC1645C** は、USB 2.0インターフェースを採用したコンパクトな1.3メガピクセル カラーCMOSカメラです。顕微鏡用途や一般的なマシンビジョン用途に使用されます。現在は生産終了（Discontinued）となっていますが、ThorCamソフトウェアやuc480ドライバを通じて制御可能です。

## 2. 基本仕様

* **型番:** DCC1645C
* **センサータイプ:** CMOS (カラー)
* **解像度:** 1280 x 1024 (SXGA)
* **最大フレームレート:** 15 fps (1280 x 1024 動作時)
    * ※ ROI (Region of Interest) やビニング設定、ピクセルクロックの調整により高速化が可能。
* **シャッター方式:** 電子ローリングシャッター (Electronic Rolling Shutter)
    * ※ 同シリーズのモノクロ版(DCC1545M)とは異なり、グローバルシャッターではありません。
* **インターフェース:** USB 2.0 (High Speed)

## 3. センサー仕様

* **有効画素数:** 1.3 Megapixels
* **光学フォーマット:** 1/2インチ
* **ピクセルサイズ:** 5.2 µm x 5.2 µm
* **スキャン方式:** プログレッシブスキャン
* **ADC:** 10-bit (8-bit出力)
* **感度:** 可視光領域（カラーフィルター配列あり）
* **赤外線カットフィルタ:** 標準搭載（取り外し可能なモデルもあり）

## 4. インターフェース・接続

* **データ/電源ポート:** USB 2.0 Mini-B コネクタ
    * USBバスパワー駆動 (外部電源不要)。
    * 消費電力: 約 1.1 W ～ 2.1 W (動作モードによる)。
* **GPIO / トリガー:** **搭載なし**
    * 本モデルは小型化優先のため、外部トリガー入力やGPIO端子（Hiroseコネクタやピンヘッダ）を物理的に搭載していません。
    * 制御は「ソフトウェアトリガー」または「フリーラン（連続取り込み）」のみとなります。ハードウェア同期が必要な用途には不向きです。

## 5. ソフトウェア・制御

PCからの制御には、Thorlabsが提供する **ThorCam** ソフトウェア、または **uc480 SDK** (旧・IDS Imaging社互換ドライバ) を使用します。

### SDK (uc480)

* **ドライバ:** `uc480.dll` / `uc480_64.dll`
* **対応言語:** C, C++, C#, VB.NET, Python, LabVIEW, MATLAB
* **主要関数例:**
    * `is_InitCamera()`: カメラ初期化
    * `is_SetImageSize()`: 解像度設定
    * `is_SetColorMode()`: カラーモード設定 (RGB24, RGB32, RAW8等)
    * `is_CaptureVideo()`: ライブキャプチャ開始
    * `is_FreezeVideo()`: スナップショット撮影

### Python制御 (PyuEye)

Pythonからは `pyueye` (IDS Imaging社のラッパー) を使用して制御可能です。

```python
from pyueye import ueye

# カメラハンドルの初期化
h_cam = ueye.HIDS(0)
ret = ueye.is_InitCamera(h_cam, None)

# カラーモード設定 (例: RGB8)
ueye.is_SetColorMode(h_cam, ueye.IS_CM_BGR8_PACKED)
```

## 6. 機械的仕様

* **レンズマウント:** Cマウント (1.00"-32)
    * CSマウントレンズを使用する場合は、付属のC-CSマウントアダプタ（5mmリング）を取り外す必要があります。
* **筐体サイズ:** 約 44.5 mm x 31 mm x 35 mm (突起部含まず、非常にコンパクト)
* **取り付け穴:**
    * 底部: 1/4"-20 (インチネジ) x 1
    * 前面: 4-40 タップ穴 x 4 (30mmケージシステム用)

## 7. 注意事項

* **生産終了品:** 本製品は生産終了しており、現行OSでのドライバサポート状況に注意が必要です（Windows 10/11ではThorCamの最新版で動作確認が必要）。
* **ローリングシャッター歪み:** 露光中に移動する被写体を撮影すると、画像が歪む「こんにゃく現象」が発生します。静止画撮影や動きの遅い観察用途に適しています。
* **USB帯域:** USB 2.0の帯域を使用するため、複数台同時接続時はフレームレートが低下する場合があります。
