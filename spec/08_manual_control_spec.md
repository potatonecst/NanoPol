# 仕様書: マニュアル制御 (Manual Control)

## 1. 概要
ユーザーがステージ（偏光子）の角度とカメラの設定を直接操作するための機能仕様です。

## 2. ステージ操作

### 2.1. Step Move (相対移動)
- **機能:** 現在地から指定した角度（Step）だけ `+` または `-` 方向に移動する。
- **UI:** `[-]` ボタン、`[+]` ボタン、ステップ量入力フィールド。
- **制約:** 移動中はUIをロックする。

### 2.2. Absolute Move (絶対移動)
- **機能:** 指定した角度（Target）へ移動する。
- **UI:** ターゲット角度入力フィールド、`[Go]` ボタン。

### 2.3. Homing (原点復帰)
- **機能:** 機械原点へ復帰し、座標を0にリセットする。
- **UI:** `[Origin]` ボタン。

### 2.4. Sweep (連続回転測定)
指定範囲を一定速度でスキャンする機能。主にプレビューや動画撮影に使用する。

- **入力:** Start角度, End角度, Speed (deg/s)
- **制約:**
  - 速度は内部的に **100PPS (Pulse Per Second) 単位** に丸められる。Zodスキーマによりバリデーションと同時に丸め込みが行われ、補正された場合はトーストとログで警告表示する。
  - Start > End の場合（逆回転）も対応する。
- **動作シーケンス:**
  1.  **速度設定:** 指定速度（100PPS単位補正済み）をステージに設定。
  2.  **助走位置へ移動:** `Start` よりも手前の `Start - Margin` へ移動。
      - `Margin` は加速距離の1.2倍（最低1.0度）を確保し、`Start` 通過時に等速であることを保証する。
  3.  **Sweep実行:** `End + Margin` へ向けて移動開始。
  4.  **自動録画 (Auto Rec):**
      - ONの場合、`Start` 通過時刻に合わせて録画を開始し、`End` 通過時刻に合わせて停止する（時間制御）。
  5.  **終了処理:** 移動完了後、速度設定を **デフォルトの高速設定** に戻す。

### 2.5. Stop (停止)
- **Stop (減速停止):** 通常の停止。移動をキャンセルし、減速して止まる。
- **Emergency Stop (非常停止):** モーター電源遮断を含む即時停止。
- **挙動:** 停止ボタン押下時は、予約されている自動録画タイマー等を即座に破棄する。

## 3. カメラ操作

### 3.1. プレビュー
- **表示:** MJPEGストリームを表示。
- **操作:** マウスホイールでズーム、ドラッグでパン移動が可能。
- **オーバーレイ:** ズーム倍率表示、リセットボタン。

### 3.2. パラメータ設定
- **Exposure (露光時間):** スライダーおよび数値入力で調整。
- **Gain (ゲイン):** スライダーおよび数値入力で調整。
- **同期:** 値変更後、0.5秒のデバウンス（待機）を経てバックエンドに送信する。

## 4. 安全機構

### 4.1. UIロック (System Busy)
- ステージ移動中やカメラ接続処理中は `isSystemBusy` フラグを立て、他の操作（画面遷移、別コマンド送信）をブロックする。

### 4.2. タイムアウト監視
- ステージ移動コマンド送信後、一定時間（デフォルト5分、Sweep時は計算値）経過しても完了しない場合はタイムアウトエラーとし、ロックを解除する。

## 5. ロギングとトレーサビリティ

- **バックエンド統合:** フロントエンドで発生した重要なイベント（操作の開始・完了、エラー、警告）は、すべて `systemApi.postLogs()` を通じてバックエンドに送信される。
- **UIとの連動:** バックエンドに出力されたログ（実機の操作記録やフロントエンドのイベント）は、画面下部の `LogPanel` でリアルタイムに確認でき、システムの完全なトレーサビリティを確保する。
